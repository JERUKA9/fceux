    FCEUX

    This is my personal branch of FCEUX, an NES/Famicom emulator. FCEUX is released under the GPL license and is available from http://www.fceux.com/ and/or http://sourceforge.net/projects/fceultra/

    I have a few goals with this branch:

    1. Better code quality
    2. Emscripten support
    3. Additional scripting features
    4. Performance

    Emscripten Demo

    This demo is a work-in-progress and does not work yet. The demo will require a new(ish) browser. It has been tested in Firefox 34.

    1. Set your command-line arguments (optional)
    2. Select a ROM on your computer
    3. Play! A:F, B:D, Select:S, Start:Enter
